Hexo 使用 Markdown 编辑技巧大全

###本地

  • 全局安装npm install hexo -g
  • 新建hexo文件夹
  • 在文件夹内部使用命令行工具 hexo init,文件夹中会生成大批文件
    • 注意此过程时间较长也许会报错 网络被墙 npm出错
  • 执行hexo g命令
  • 执行hexo s命令
    • 此命令会默认打开一个小型本地服务器 通过http://localhost:4000/可访问 文件夹中 source 下_posts中的文件 本地预览文件
    • 注意端口不要被占用
  • 试着在source 下_posts文件夹里 新建文件hexo.md 并在里面写内容
    • 注意文件编码使用 utf-8

上传

  • 执行命令 hexo g

    • 注意此命令是把md文件生成HTML
    • 如果有缓存 可以执行命令 hexo clean后再hexo g

  • 打开GitHub.com 登录并新建仓库

    • 注意次仓库的名称必须是以下格式 例如xxx.github.io xxx是你GitHub的用户名

  • 执行命令 npm install hexo-deployer-git --save

  • 执行hexo d 命令 推送至GitHub上 通过仓库,名即可访问
